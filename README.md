# Sistema de Mercadorias de Bebidas
Este projeto é um sistema simples de controle de estoque de bebidas para distribuidoras.  
Permite cadastrar bebidas, movimentar estoque (entrada/saída) e gerar relatórios com alerta de estoque baixo.

Funcionalidades
- Cadastro de bebidas (nome, categoria, preço, fornecedor, quantidade).
- Listagem de bebidas cadastradas.
- Movimentação de estoque (entrada e saída).
- Relatório de estoque com alerta de baixo nível
